# Rock Paper Scissors Game

A simple desktop game built using **Python** and **Tkinter** where the user plays Rock, Paper, Scissors against the computer. The computer makes random choices, and the game instantly displays the result.

## Features

- Interactive GUI using Tkinter
- Random computer opponent
- Win, lose, and tie detection
- Simple and beginner-friendly design

## Technologies Used

- Python
- Tkinter
- Random Module

## Learning Outcomes

- Python fundamentals
- GUI development with Tkinter
- Event handling
- Conditional statements
- Random module
